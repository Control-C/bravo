AI projects demo

Visit 2 proof-of-concept projects that use Gemini AI:

  1. Tool: ‘Umbra’ is a tool that redacts personally identifiable information on your local machine using Gemini Nano, before information is sent to Gemini cloud services.
  2. Use case: ‘Bravo’ shows an example of how the Umbra tool can be used, in the context of a hospital consultation.

Step 1

Confirm that you are using the Chrome browser on a desktop computer (the demos are made for wide screens). If you are, then go to step 2.

Step 2

The demo projects need a key for the Google Gemini API. You can sign up for an API Key at Google (requires a Google account)

If you don't have an API key, the demos will not work properly. Watch a pre-recorded video demo on You Tube instead.




Step 3

Which project would you like to visit?

Umbra

Umbra is a tool that prevents your private data from being shared to online services. It uses Gemini Nano to redact personal, sensitive, or confidential information on your local machine. Your redacted data is then sent off to Gemini cloud services for a response. When the response is received, Umbra replaces the redated data with the original texts.

Bravo

Bravo demonstrates how hospitals can use AI to build a privacy-first healthcare system that's efficient for doctors and delightful for patients.

Using the Umbra privacy tool, the Bravo software streamlines a doctor's workflow during a patient consultation. A medical document to be sent to the patient's insurance company, personalized to the patient's conditions, is quickly generated using AI, without passing personally identifiable information about the patient to third party cloud services. At the same time, the same data is used to create a personalized story book about their illness for the young patient.

‘Bravo’ is a fictitious use case, but the scenario of a doctor using patient information to ask an LLM to write an insurance letter was created from a real-world incident (source).
