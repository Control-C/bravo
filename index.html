AI projects demo

Explore 2 proof-of-concept project demonstrations that use Gemini AI:

  1. Tool: ‘Umbra’ is a privacy-first tool that redacts Personal Identifiable Information (PII) on your local machine using Gemini Nano, before information is sent to Gemini cloud services.
  2. Use case: ‘Bravo’ shows an example of how the Umbra tool can be used, in the context of a medical consultation at a children's hospital.

Step 1

Confirm that you are using the Chrome browser on a desktop computer (the demos are made for wide screens).

If confirmed, proceed to step 2.

Step 2

The demo projects need a key for the Google Gemini API. You can sign up for an API Key at Google (requires a Google account)

If you don't have an API key, the demos will not work properly. You can watch a pre-recorded video demo on YouTube instead.

Step 3

Choose a project:

Umbra

Umbra is a privacy-first tool that prevents your private data from being shared to online services.

Umbra uses Gemini Nano to redact PII on your local machine. Your redacted data is then sent off to Gemini cloud services for a response. When the response is received, Umbra replaces the redated data with the original texts.

Bravo

Bravo demonstrates how hospitals can use AI to build a privacy-first healthcare system that's efficient for doctors and delightful for patients.

Using the Umbra privacy tool, the Bravo software streamlines a doctor's workflow during a patient consultation. The patient's medical data with redacted PII is used by Gemini AI to:

  • For the doctor: write a letter to the patient's insurance company justifying the medical procedures.
  • For the 5-year-old patient: write a story to help her better understand her medical condition, personalized using a favorite toy as the central character.
‘Bravo’ is a fictitious use case, but the scenario of a doctor disclosing patient information to ask a Large Language Model to write an insurance letter was created from a real-world incident (source).
